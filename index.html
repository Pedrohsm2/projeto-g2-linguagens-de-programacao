🏥 Projeto G2 — Saúde Pública no Brasil

Análise Exploratória de Indicadores de Saúde Pública utilizando Python, Pandas, Seaborn e Streamlit

📌 Sobre o Projeto

Este projeto tem como objetivo analisar indicadores de saúde pública no Brasil entre os anos de 2015 e 2024.


Foram avaliadas informações relacionadas à expectativa de vida, mortalidade, vacinação, internações e infraestrutura hospitalar, buscando identificar padrões, tendências e diferenças regionais.

🎯 Objetivos

  • Analisar indicadores de saúde pública brasileiros.
  • Comparar regiões do país.
  • Identificar tendências temporais.
  • Avaliar relações entre vacinação e mortalidade.
  • Construir visualizações interativas para apoio à análise.

📊 Indicadores Analisados

❤️ Expectativa de Vida
📉 Taxa de Mortalidade
💉 Cobertura Vacinal
🏥 Leitos Hospitalares
👨‍⚕️ Médicos por 1000 Habitantes
📋 Taxa de Internação

🛠 Tecnologias Utilizadas

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • Streamlit
  • Git
  • GitHub

📈 Principais Resultados

  • Identificação de diferenças regionais nos indicadores de saúde.
  • Relação entre cobertura vacinal e mortalidade.
  • Análise da evolução temporal da expectativa de vida.
  • Avaliação da infraestrutura hospitalar por região.
  • Construção de dashboard interativo para exploração dos dados.
