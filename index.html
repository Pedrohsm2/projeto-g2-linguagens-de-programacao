Projeto G2 — Saúde Pública no Brasil

Análise de Indicadores de Saúde Pública utilizando Python e Streamlit

Objetivo

Este projeto realiza uma análise exploratória de indicadores de saúde pública no Brasil entre 2015 e 2024. Foram avaliados indicadores como expectativa de vida, mortalidade, cobertura vacinal, internações e infraestrutura hospitalar.

Tecnologias Utilizadas

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • Streamlit
  • GitHub
