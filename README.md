# 🏥 Projeto G2 — Saúde Pública no Brasil

## 📌 Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de Linguagens de Programação com o objetivo de realizar uma análise exploratória de indicadores de saúde pública no Brasil entre os anos de 2015 e 2024.

A análise busca identificar padrões, tendências temporais e diferenças regionais relacionadas à expectativa de vida, mortalidade, cobertura vacinal, internações e infraestrutura hospitalar.

---

## 🎯 Objetivos

- Analisar indicadores de saúde pública no Brasil.
- Identificar tendências ao longo do tempo.
- Comparar regiões brasileiras.
- Avaliar relações entre vacinação, mortalidade e infraestrutura hospitalar.
- Construir um dashboard interativo para visualização dos dados.

---

## 📊 Indicadores Analisados

- Expectativa de vida
- Taxa de mortalidade
- Taxa de internação
- Cobertura vacinal
- Leitos hospitalares
- Médicos por 1000 habitantes
- Casos de doenças crônicas
- Nível de criticidade
